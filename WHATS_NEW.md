## 💡 Example Conversations

### Student asks about Binary Numbers:
**Student:** "I don't understand what binary numbers are"

**AI Tutor:** "Binary numbers are a way of representing numbers using only two digits: 0 and 1. Think of it like a light switch - it can only be ON (1) or OFF (0). In our regular decimal system, we use 10 digits (0-9), but computers use binary because electronic circuits can easily represent two states. For example, the number 5 in decimal is 101 in binary..."

### Student needs clarification:
**Student:** "Can you give me a simpler example?"

**AI Tutor:** "Sure! Imagine you have 3 light switches in a row. Each switch can be either ON or OFF. If all switches are OFF, that's 000 in binary (which equals 0). If only the last switch is ON, that's 001 in binary (which equals 1). If the middle switch is ON, that's 010 in binary (which equals 2). This is how computers count!"
